# Epilepsy-Detection-using-Online-Clustering
A Haskell Project made to identify EEG spikes using online clustering detection
This Haskell Project is made for an EEG data which identifies spikes in the EEG graph finding abnormalities that detect epilepsy.
How to Use:
1. Download the eeg dataset.
2. Run the main.hs file and this will create a csv files where distance gets saved.
3. Run the HTML file and upload the distance.csv which will then display the graph along with numerical interpretation of the given data.
